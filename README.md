# UG-rPPG — Official Implementation

Official PyTorch code for **Uncertainty-Guided Remote Photoplethysmography for Robust Pulse Estimation** (**UG-rPPG**).

UG-rPPG integrates an embedded FANLayer into a bidirectional Mamba backbone for temporal-frequency feature perception, and introduces an evidential deep learning head for per-point waveform uncertainty estimation. A dynamic-weight joint optimization strategy is further used to balance waveform regression and evidential uncertainty learning.:

- **FANLayer** — frequency-aware feature refinement
- **EDL head** — evidential uncertainty estimation (aleatoric + epistemic)
- **Dynamic-weight training** — adaptive balance between waveform loss and evidential loss

This repository contains the configs, scripts, and pretrained weights needed to reproduce the paper experiments.

## Tested environment

Paper experiments and bundled checkpoints were produced with:

| Component | Version |
|-----------|---------|
| Python | 3.8 |
| PyTorch | 1.13.0 |
| torchvision | 0.14.0 |
| torchaudio | 0.13.0 |
| CUDA | 11.x (GPU required for training / testing) |
| causal-conv1d | 1.2.0.post2 |
| mamba-ssm | 1.2.0.post1 |

Newer Python / PyTorch stacks are untested and may fail to install `mamba-ssm` or change numerical results.

---

## Environment setup

```bash
cd UG-rPPG

# 1) Create env (conda example)
conda create -n ugrppg python=3.8 -y
conda activate ugrppg

# 2) Install PyTorch (adjust pytorch-cuda for your driver)
conda install pytorch==1.13.0 torchvision==0.14.0 torchaudio==0.13.0 pytorch-cuda=11.7 -c pytorch -c nvidia -y

# 3) Install remaining Python dependencies
pip install -r requirements.txt

# 4) Install Mamba stack (CUDA-specific wheels; versions must match PyTorch + CUDA)
pip install causal-conv1d==1.2.0.post2
pip install mamba-ssm==1.2.0.post1
```

If step 4 fails, install pre-built wheels matching your CUDA toolkit, or build from source following the [mamba-ssm](https://github.com/state-spaces/mamba) instructions.

Verify syntax:

```bash
python -m compileall .
```

## Dataset preparation

1. Download **PURE**, **UBFC-rPPG**, and **MMPD** following the original dataset instructions.
2. Edit the corresponding yaml file: set `DATA_PATH` (raw) and `CACHED_PATH` (preprocessed cache).
3. First run: set `DO_PREPROCESS: true` for train/valid/test splits.
4. Later runs: set `DO_PREPROCESS: false` to load cached `.npy` clips.

Face detection uses `dataset/haarcascade_frontalface_default.xml` (OpenCV Haar cascade).
